== Plugin Description ==

How to use  the plugin:

1. Install by upload the package file or upload the folder into wp-content/plugins directory.
2. Activate the Mobile browser color select Plugin.
3. Under `Settings` tab inside the Admin control panel menu
Click `Mobile browser color select`
4. Choose your favorite color
5. Hit `Save`.
 - That`s it. Done.
